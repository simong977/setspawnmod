SetSpawnMod (for Minecraft 1.21.1 + NeoForge 21.1.113)

이 모드는 /setspawn 과 /spawn 명령어를 추가합니다.
 - /setspawn : 현재 위치를 서버 전체의 스폰 위치로 저장
 - /spawn    : 저장된 위치로 즉시 텔레포트

✅ 빌드 방법:
1. IntelliJ 또는 Eclipse에서 Gradle 프로젝트로 열기
2. Gradle 탭에서 'build' 실행하거나, 터미널에서:
   ./gradlew build
3. 빌드 결과는 다음 경로에 생성됨:
   build/libs/setspawnmod-1.0.jar

📦 빌드된 .jar 파일이 필요하면 ChatGPT에게 요청하세요. (AI는 직접 JAR 생성은 제한됨)
